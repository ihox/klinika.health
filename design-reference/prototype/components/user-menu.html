  • .user-menu — container, 232px min-width, elevated shadow, dropdown nën avatar.
  • .um-head — bllok identiteti me bg-subtle dhe border-bottom. Përmban .name, pastaj ose .email + .roles (klinikë) ose .role (platform admin).
  • .um-item — rresht meny me .ic (16px) majtas + label. Hover = bg-subtle.
  • .um-item.logout — variant i kuq për logout. Hover = red-bg.
  • .um-divider — vijë e hollë midis grupeve të zërave (gjithmonë para logout).
